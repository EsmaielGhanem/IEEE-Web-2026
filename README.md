Database Fundamentals – IEEE MSB

This repository contains the materials and sessions delivered as part of the Web Track at IEEE MSB, covering essential database concepts and SQL fundamentals.

📌 Overview

The goal of this content is to introduce students to core database concepts and provide hands-on experience with SQL and relational database design.

🧠 Topics Covered
🔹 SQL Basics
CRUD Operations:
SELECT
INSERT
UPDATE
DELETE
Filtering:
WHERE
LIKE
IN
BETWEEN
Sorting:
ORDER BY
🔹 Database Relationships
One-to-One
One-to-Many
Many-to-Many
Foreign Key Concept
🔹 Joins
INNER JOIN
LEFT JOIN
RIGHT JOIN
🔹 Advanced Database Concepts
Stored Procedures
Functions
Aggregate Functions
Views
Triggers
Indexes
⚙️ Practical Approach
Writing real SQL queries
Designing relational schemas
Applying best practices in database design
Understanding performance basics using indexing
🎯 Learning Outcomes

By the end of these sessions, students are able to:

Write efficient SQL queries
Design relational databases
Understand and implement relationships between entities
Use advanced database features in real-world scenarios
