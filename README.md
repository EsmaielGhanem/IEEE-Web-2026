# Database Fundamentals – IEEE MSB

## 📌 Overview

This repository contains the materials and sessions delivered as part of the Web Track at IEEE MSB. It focuses on building a strong foundation in SQL and relational database concepts with practical examples.

---

## 🧠 Topics Covered

### 🔹 SQL Basics

* **CRUD Operations**

  * SELECT
  * INSERT
  * UPDATE
  * DELETE

* **Filtering**

  * WHERE
  * LIKE
  * IN
  * BETWEEN

* **Sorting**

  * ORDER BY

---

### 🔹 Database Relationships

* One-to-One
* One-to-Many
* Many-to-Many
* Foreign Key Concept

---

### 🔹 Joins

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN

---

### 🔹 Advanced Concepts

* Stored Procedures
* Functions
* Aggregate Functions
* Views
* Triggers
* Indexes

---

## ⚙️ Practical Work

* Writing real SQL queries
* Designing relational database schemas
* Applying best practices
* Understanding indexing and performance basics

## 🎯 Learning Outcomes

* Ability to write efficient SQL queries
* Understanding relational database design
* Implementing relationships between entities
* Using advanced database features in real scenarios
